--------------------------------------------------------------------------------
EXECUTIVE SUMMARY
--------------------------------------------------------------------------------
P20-B delivers the first operational workforce attendance event foundation:
canonical tables, ingestion/normalization/summary services, transitional
dual-write to hr_attendance, web clock APIs, event bus hooks, and parallel
excel import ingestion—without changing report read paths or removing legacy
attendance APIs.
