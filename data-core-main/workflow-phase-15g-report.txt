================================================================================
6) COMMERCIAL ACTIVITY — DECISION
================================================================================

NOT implemented as live feed in P15-G.

Reason:
  GET /platform/activity filters workspaceId IS NULL (platform-user events only).
  Commercial audit events are written with workspaceId set on tenant actions.
  A tenant-scoped commercial activity API would require new backend scope (deferred).

UI shows read-only notice (data-testid=commercial-activity-unavailable) when user has
activity permission, explaining the limitation.
