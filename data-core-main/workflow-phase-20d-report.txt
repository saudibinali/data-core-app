Workforce Self-Service Clocking & Geofence Foundation
================================================================================
Date: 2026-05-19
Scope: P20-D (no P20-E)

--------------------------------------------------------------------------------
EXECUTIVE SUMMARY
--------------------------------------------------------------------------------
P20-D delivers employee web self-service clocking with geofence and policy
foundation (warning-first), enriched location metadata on canonical events,
self-service APIs/UI, and reporting column extensions—without mobile apps,
live GPS tracking, vendor SDKs, or biometric devices.

--------------------------------------------------------------------------------
SELF-SERVICE STATUS
--------------------------------------------------------------------------------
APIs:
- GET /hr/workforce/me/status
- GET /hr/workforce/me/today
- GET /hr/workforce/me/history
- POST /hr/workforce/clock-in | clock-out (enhanced)

UI:
- /self-service/attendance (hr-me-attendance.tsx)
- Self-service hub quick link

Status: GO

--------------------------------------------------------------------------------
GEOFENCE FOUNDATION STATUS
--------------------------------------------------------------------------------
Tables: attendance_geofences (circle radius, work_location_id link)
Service: GeofenceValidationService (haversine + accuracy buffer)
Admin APIs: GET/POST /hr/workforce/geofences
Mode: Warning-first (no hard reject by default)

Status: GO (circle only; no polygons/maps)

--------------------------------------------------------------------------------
POLICY SYSTEM STATUS
--------------------------------------------------------------------------------
Table: attendance_policies
Fields: geofenceRequired, allowRemoteClock, suspiciousLocationAction,
        minAccuracyMeters, graceMinutes, duplicateWindowSeconds
Default seed on init per workspace
Admin APIs: GET/POST /hr/workforce/policies

Status: GO

--------------------------------------------------------------------------------
PRIVACY / SECURITY STATUS
--------------------------------------------------------------------------------
- Punch-time-only location (UI opt-in)
- No background tracking
- Workspace isolation on all queries
- Warnings in raw payload (employee-visible via status API)
- Access logging hooks on clock actions
- Manager location export not expanded (by design)

Status: GO (foundation); PARTIAL (no dedicated audit table)

--------------------------------------------------------------------------------
REPORTING COMPATIBILITY
--------------------------------------------------------------------------------

Added export columns: Source, Geofence Flag

Status: GO
