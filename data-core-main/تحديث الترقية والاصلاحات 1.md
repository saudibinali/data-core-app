### **اختياري**
- `REDIS_URL` لتفعيل Redis (مع fallback عند عدم التوفر)
- `DATABASE_READ_URL` لتفعيل read replica
- `S3_*` لتفعيل MinIO/S3-compatible object storage
- `METRICS_ENABLED=false` لتعطيل `/api/health/metrics`
