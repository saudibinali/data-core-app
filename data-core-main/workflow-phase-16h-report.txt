  isAutomaticAllowed=false — no auto-apply (tests: commercial-workspace-enforcement-evaluator).
  Super Admin: manual PATCH workspace-access with reason (platform.workspaceAccess.update).
  Tenant: readOnlyMode true after apply; banner + subscription page banner.
  GET: tenant subscription + billing exempt from write guard.
  POST/PATCH/DELETE: blocked by workspaceAccessWriteGuard (resolver tests + middleware).

  PASS (design): recommendation suspended_view_only; manual apply only.
  Tenant: allowLogin/allowRead true; create/update/delete false.
  Billing: /tenant/billing/* exempt — invoice list/PDF per tenant.billing.* permissions.
  No purge/delete routes in Phase 16.

Scenario D — Terminated
  PASS (design): recommendation terminated_view_only; manual apply.
  allowLogin remains true in TERMINATED_VIEW_ONLY_FLAGS (no default full login block).
  allowAdminAccess false in terminated_view_only preset.
  No destructive workspace actions in Phase 16 APIs.
