عنوان المهمة:
WORKFLOW PHASE 14-D — Platform User Audit & Activity Tracking

المهمة الحالية:
بناء قراءة منظمة لنشاط مستخدمي المنصة ومحاولات الوصول والصلاحيات، اعتمادًا على activity_logs الحالية، بدون تعديل أو حذف سجلات.

الهدف:
تمكين Platform Owner / authorized roles من رؤية:
- من أنشأ Platform User
- من غيّر حالة Platform User
- من غيّر دور Platform User
- من مُنع بسبب صلاحية ناقصة
- من حاول إجراء محظور على Root/Protected account
- من نفذ أو حاول إجراءات حساسة على Tenant lifecycle/subscription/entitlements
- متى حدث ذلك
- النتيجة success/blocked/denied
- السبب reason / blockedReason

بطريقة:
read-only platform audit visibility
بدون بناء نظام SIEM وبدون تعديل السجلات

أضف config/lib:

platform-audit-events.ts

Event groups:
- platform_user_management — إدارة مستخدمي المنصة
- platform_role_management — إدارة أدوار المنصة
- platform_permission_denial — رفض الصلاحيات
- root_protection — حماية المالك الجذري
- tenant_lifecycle — دورة حياة المستأجر
- tenant_subscription — اشتراك المستأجر
- tenant_entitlement — صلاحيات وميزات المستأجر
- platform_access — وصول المنصة

Event config:
لكل event:
- actionCode
- label
- labelAr
- group
- severity:
  - info
  - warning
  - critical
- resultType:
  - success
  - blocked
  - denied
  - failed
- description

Include known events:
- platform_user_created — إنشاء مستخدم منصة
- platform_user_create_blocked — منع إنشاء مستخدم منصة
- platform_user_status_changed — تغيير حالة مستخدم منصة
- platform_user_status_change_blocked — منع تغيير حالة مستخدم منصة
- platform_user_role_changed — تغيير دور مستخدم منصة
- platform_user_role_change_blocked — منع تغيير دور مستخدم منصة
- platform_permission_denied — رفض صلاحية منصة
- protected_root_action_blocked — منع إجراء على المالك الجذري
- platform_user_access_policy_violation — مخالفة سياسة وصول مستخدم منصة
- tenant_lifecycle_changed — تغيير دورة حياة مستأجر
- tenant_subscription_updated — تحديث اشتراك مستأجر
- tenant_entitlement_override_updated — تحديث استثناء صلاحيات مستأجر

Unknown events:
- group = platform_access
- severity = info
- label = actionCode
- labelAr = "حدث غير مصنف"

- PLATFORM_AUDIT_EVENT_GROUPS

- isPlatformAuditEvent(actionCode)

أضف APIs:

GET /platform/activity

Permission:
- requirePlatformPermission("platform.activity.read")
أو audit.read

Query params:
- actorId optional
- targetUserId optional
- action optional
- group optional
- result optional
- severity optional
- from optional ISO date
- to optional ISO date
- limit default 50 max 200
- cursor optional

Returns:
- items[]
  - id
  - actorId
  - actorEmail/displayName if join available
  - targetUserId
  - targetEmail/displayName if join available
  - action
  - actionLabel
  - actionLabelAr
  - group
  - severity
  - result
  - reason
  - blockedReason
  - resourceType
  - resourceId
  - metadataSafe
  - createdAt
- nextCursor

Rules:
- read-only
- no secrets
- redact sensitive metadata keys:
  - password
  - token
  - secret
  - apiKey
  - authorization
  - cookie
- stable pagination
- newest first

GET /platform/users/:userId/activity

Permission:
- requirePlatformPermission("platform.activity.read")
أو audit.read

Returns latest activity for a specific platform user:
- as actor
- or target

Query:
- limit default 20 max 100
- cursor optional

Rules:
- read-only
- cannot expose secrets
- should include blocked attempts
