راجع الـ EventBus architecture الحالية بالكامل ثم نفّذ فقط مهمة تحليل وتصميم Observability Layer بدون أي implementation فعلي حتى الآن.

عنوان المهمة:
EVENT OBSERVABILITY & DIAGNOSTICS — Architecture Review

المهمة الحالية:
تحليل جميع طبقات:
- event logging
- tracing
- diagnostics
- workflow visibility
- listener execution visibility
- operational monitoring

داخل النظام الحالي قبل بناء أي observability infrastructure جديدة.

الهدف:
تصميم Enterprise-grade Event Observability Layer
بدون تكرار وظائف موجودة أو خلق logging chaos.
