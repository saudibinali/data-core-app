3 — USAGE STATUS MODEL
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

أضف:

UsageLimitStatus:
- unknown
- normal
- approaching
- exceeded
- unlimited
- not_applicable

Rules:
- unknown: usage غير متوفر أو limit غير معروف
- unlimited: limit null ومعناه unlimited
- not_applicable: metric غير مستخدم لهذه الخطة
- normal: < 80%
- approaching: >= 80% and < 100%
- exceeded: >= 100%

مهم:
thresholds تكون constants:
- USAGE_APPROACHING_THRESHOLD = 0.8
- USAGE_EXCEEDED_THRESHOLD = 1.0

5 — RAW USAGE COLLECTION
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

أضف function:

يحاول يحسب:
- seats: count active users in workspace إن كان واضح
- documents: count documents إن كان table موجود
- storage_gb: إذا موجود field استخدمه، وإلا unknown
- monthly_api_calls: unknown إذا لا يوجد counter
- workflows: count إن وجد
- integrations: count إن وجد
- ai_actions: unknown
- custom_reports: unknown
- audit_retention_days: configured من limits وليس usage فعلي
- workspaces: 1 غالبًا

مهم:
- لا تضف jobs
- لا تعمل queries ثقيلة
- استخدم counts بسيطة فقط
- unknown أفضل من حساب خاطئ
