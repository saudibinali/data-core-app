3 — EVALUATION DERIVATION LIB
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Exports:
- deriveLifecycleEvaluationSignals(input)
- deriveEvaluationSeverity(signals)
- deriveEvaluationRecommendedAction(signals, severity)
- deriveReviewEligibility(signals, severity)
- deriveTenantLifecycleEvaluationProfile(input)
- buildEvaluationWarnings(profile)
- buildEvaluationSummary(profile)

TenantLifecycleEvaluationInput:
- tenantProfile
- lifecycleState
- subscriptionStatus
- renewalProfile
- usageProfile
- entitlementProfile
- healthProfile
- governanceRiskSummary optional

TenantLifecycleEvaluationProfile:
- tenantId
- workspaceId
- signals[]
- severity
- recommendedAction
- reviewEligibility
- warnings[]
- summary
- evaluatedAt
- safetyNotice

Rules:
- critical health → critical severity + manualReviewRequired
- grace expired → critical + suspensionReviewEligible true لكن بدون suspend
- subscription expired → high + renewalReviewEligible true
- renewal due now/soon → medium + renewalReviewEligible true
- usage exceeded → high + usageReviewEligible true
- usage approaching → medium + usageReviewEligible true
- entitlement overrides → low/medium + entitlementReviewEligible true
- workspace suspended/locked → high + lifecycleReviewEligible true
- governance warning → medium/high + governanceReviewEligible true
- missing data → unknown/medium + manualReviewRequired true

مهم:
لا يوجد mutation.
لا يوجد تغيير workspace.status.
لا يوجد enforcement.
