6 — ENTITLEMENT DERIVATION LIB
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

tenant-entitlements.ts

Exports:
- derivePlanEntitlements(planCode)
- applyEntitlementOverrides(base, overrides)
- deriveTenantEntitlementProfile(planCode, overrides)
- isModuleEnabled(profile, moduleCode)
- getFeatureLimit(profile, limitCode)
- validateEntitlementOverride(input)
- buildEntitlementAuditPayload(...)

EntitlementProfile:
- planCode
- enabledModules
- disabledModules
- limits
- overridesApplied
- customEntitlementsCount
- derivedAt

Validation:
- moduleCode valid
- limitCode valid إذا override limit
- reason min length 10
- enable/disable cannot target unknown module
- limit_override requires limitCode + limitValue
- limitValue must be non-negative or null if unlimited allowed
- cannot disable core safety module إذا قررت تمنعه

7 — BACKEND API
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

أضف APIs:

GET /platform/tenants/:tenantId/entitlements

PATCH /platform/tenants/:tenantId/entitlements/overrides

Body:
{
  "overrides": [
    {
      "moduleCode": "payroll",
      "overrideType": "enable",
      "limitCode": null,
      "limitValue": null,
      "reason": "..."
    }
  ],
  "confirmation": true
}

Behavior:
- requireAuth
- requireSuperAdmin
- validate tenant exists
- read subscription planCode
- derive base entitlements
- apply overrides
- PATCH upserts overrides
- audit log event:
  tenant_entitlements_updated
- return updated entitlement profile
