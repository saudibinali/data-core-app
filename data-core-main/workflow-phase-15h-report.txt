3) COMMERCIAL ACTIVITY DECISION
================================================================================

Decision: IMPLEMENTED (safe, read-only, tenant-scoped).

Endpoint:
  GET /platform/tenants/:tenantId/commercial-activity
  Permissions: platform.activity.read OR audit.read (requireAnyPlatformPermission)
  Super-admin required
  Default limit: 30, max: 50
  Filters: workspaceId = tenantId AND action LIKE 'commercial_%'
  Post-filter: commercial_account_, commercial_billing_contact_, commercial_contract_,
               commercial_invoice_, commercial_payment_, commercial_risk_, commercial_access_denied

Response fields per item:
  id, action, actionLabel, actionLabelAr, severity, result,
  actorId, actorDisplayName, metadataSummary (safe keys only), createdAt

Safety:
  - Uses parseAndRedactMetadata via enrichRow (no secrets)
  - No audit log on read (avoids noise)
  - GET only — no POST/PATCH/DELETE

Frontend:
  Commercial tab → Commercial Activity accordion → list with loading/empty/error states
