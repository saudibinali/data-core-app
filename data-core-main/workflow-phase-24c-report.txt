P24-C — Procurement Workspace UI & Operations Center

Executive summary
-----------------
P24-C converts the Procurement canonical foundation (P24-B) into a usable workspace runtime module by adding:
- Workspace-scoped Procurement APIs (vendors, PR, RFQ, PO, policies, ops metrics)
- Workspace UI pages under /procurement (Vendors / PR / RFQ / PO / Policies / Reports / Ops Center)
- Procurement lifecycle events + notification wiring
- Procurement monitoring/ops overview endpoint

All work follows the constraints:
- No inventory rollout
- No AP invoice engine
- No payment execution
- No supplier portal
- No accounting posting
- No destructive migrations
- No external procurement integrations
