EXECUTIVE SUMMARY
--------------------------------------------------------------------------------
Implemented a vendor-agnostic Workforce Integration Hub atop the P20-B event
platform: schema migrations for integrations/devices/employee maps, plugin-style
ConnectorRegistry + AttendanceConnector contract, four minimal connectors, webhook
ingestion with signature/replay/idempotency, DB-backed polling sync worker (no
Redis), management APIs, security primitives, notification hooks, smoke tests,
and documentation.

Constraints honored: no vendor SDK lock-in, no payroll redesign, no destructive
migration, no biometric template storage, no mobile app, no Redis/BullMQ redesign.
